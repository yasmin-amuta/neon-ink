# Neon Ink

Neon Ink is a bold, dark-mode agency website for a branding and marketing studio focused on turning ideas into powerful, immersive brand experiences.
The site showcases the agencyâ€™s philosophy, services, and work through a highly visual, animation-driven interface that blends minimal design with expressive motion.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, modern layout techniques)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- ScrollTrigger (for scroll-based animations)

## Features

- Responsive layout optimized for desktop, tablet, and mobile
- Animated hero section with inline video typography
- Custom navigation with interactive pill menu
- Scroll-based animations (fade-ins, overlaps, parallax effects)
- Interactive SVG elements and cursor-based micro-interactions
- Dark-mode, minimal, editorial-style layout
